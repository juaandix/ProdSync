# ProdSync — Roadmap de mejoras

## 1. Experiencia de usuario

- [x] Búsqueda global en el header con resultados instantáneos (proyectos, clientes, tareas)
- [x] Filtros y ordenación en tabla de proyectos
- [x] Skeleton loaders consistentes en toda la app
- [x] Perfil de usuario editable (nombre, contraseña)

## 2. Datos y reporting

- [x] Informes de tiempo — horas por proyecto, usuario, tipo y período con filtros
- [x] Exportar a CSV — descarga directa desde la página de informes
- [x] Rentabilidad de proyectos — horas invertidas vs presupuesto
- [x] Dashboard por rol — vistas distintas para ADMIN, OPERATOR y USER

## 3. Gestión avanzada

- [x] Presupuestos con cálculo real — conectar horas registradas con líneas de presupuesto
- [x] Historial de actividad — quién registró qué y cuándo, por proyecto y tipo
- [x] Estados de proyecto automáticos — cambiar a COMPLETADO cuando todas las tareas están hechas

## 4. Calidad comercial

- [x] Página de error y estados vacíos con diseño cuidado
- [x] Textos 100% en español en toda la app
- [ ] Responsive móvil
